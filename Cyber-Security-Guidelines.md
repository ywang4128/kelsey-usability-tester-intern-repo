# Cyber Security
## What security measures do you currently follow, and where can you improve?
The security measures i'm currently using are:  
- keep my devices updated
- use 2FA on my student account
- being cautious when opening links or downloading files
- using my own hotspot instead of connecting to unsecure public wi-fi

Areas I can improve:
- ensure all my accounts use strong and unique passwords
- enable two-factor authentication on more platforms


## How can you make secure behaviour a habit rather than an afterthought?
to make the secure behaviour a habit, i think the best wat is by integrating it into daily life and using tools that automate security.
e.g. lock the computer every time i step away, double check links before clicking, etc.


## What steps will you take to ensure your passwords and accounts are secure?
- use strong and unique passwords for each account
- enable 2FA to each platform
- store passwords in a secure password manager
- avoid sharing passwords or storing them in insecure compromise


## What would you do if you suspected a security breach or suspicious activity on your account?
I'll change the password for that account, check recent login activity for unauthorized access, and report the issue to the relevant IT support or service provider.

use 2FA on my working account
